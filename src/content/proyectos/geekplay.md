**Resumen:** GeekPlay es un foro de tecnología pensado para crecer de forma modular. Separé el dominio en servicios independientes para usuarios, autenticación y publicaciones, con énfasis en una API clara y mantenible.

**Responsabilidades clave:**
- Diseño de endpoints REST y validaciones.
- Modelado de base de datos y consultas SQL.
- Integración frontend con React y manejo de estados.

**Resultados:**
- Arquitectura preparada para escalar por servicio.
- Mejor separación de responsabilidades y mantenibilidad.
- Base sólida para futuras funcionalidades (moderación, notificaciones).
